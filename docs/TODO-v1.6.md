┌─────────────────────────────────────┐
│  数据库双重备份机制                  │

### 5. 阿里云个人服务器部署 ⭐ 高优先级
**状态**: 🔲 待开发  
**说明**: 部署到个人阿里云服务器，以网页形式对外服务

**部署方案**:
```
用户访问流程:
用户 → 阿里云域名/公网IP → 轻量应用服务器
                                    ├── Nginx (前端)
                                    ├── Node.js + PM2 (后端)
                                    └── MySQL (数据库)
```

**服务器配置推荐**:
| 项目 | 配置 | 预估费用 |
|------|------|---------|
| 轻量应用服务器 | 2核4G, 60GB SSD, 5M带宽 | ~100元/月 |
| 系统镜像 | CentOS 8 / Ubuntu 20.04 | - |
| 域名（可选） | .com/.cn 域名 | ~50元/年 |

**部署步骤**:
- [ ] 购买阿里云轻量应用服务器
- [ ] 配置安全组（开放80, 443, 3001端口）
- [ ] 安装环境：Node.js 18 / MySQL 8 / Nginx / PM2
- [ ] 从GitHub拉取最新代码
- [ ] 导入数据库结构和初始数据
- [ ] 配置后端环境变量（数据库连接等）
- [ ] 构建前端静态文件
- [ ] 配置Nginx反向代理
- [ ] 使用PM2启动后端服务
- [ ] 配置域名解析（可选）
- [ ] 申请SSL证书并配置HTTPS（可选）

**自动化脚本**:
- [ ] 编写一键部署脚本 `deploy/aliyun-deploy.sh`
- [ ] 编写环境初始化脚本 `deploy/setup.sh`
- [ ] 编写备份脚本 `deploy/backup.sh`

**部署文档**:
- [ ] 编写详细的部署教程 `DEPLOYMENT-ALIYUN.md`

|------|------|-----------|---------|

| 库存正常 | 绿色 #67C23A | /ppe-list?filter=normal | quantity > safety_stock |
| 库存偏低 | 黄色 #E6A23C | /ppe-list?filter=low | 0 < quantity <= safety_stock |
